# **App Name**: VesselTrack

## Core Features:

- Vessel Number Input: Allow users to input a vessel number for tracking.
- Real-time Tracking: Dynamically display the vessel's real-time location and status from a free shipping vessel tracker website.
- User Authentication: Enable users to create accounts and log in using their Google accounts.
- AI-Powered Route Analysis: Use AI to provide estimated time of arrival and potential route optimizations; uses shipping data as a tool to inform the reasoning process.

## Style Guidelines:

- Primary color: Dark blue (#1A237E) to evoke trust and maritime themes.
- Background color: Light gray (#E8EAF6) to ensure readability and a clean interface.
- Accent color: Teal (#008080) for interactive elements and calls to action.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, creating a balance of modernity and readability.
- Use clear and concise icons related to shipping and navigation, with a modern, minimalist style.
- Design a clean and intuitive layout that prioritizes real-time vessel tracking information.
- Subtle animations to indicate loading and updating of vessel location data.
